# nvim_config
My take on configuring neovim using lua. Requires version 0.5.0+
Mostly check version of telescope, newer versions need 0.6.0 to work.


## Currently used plugins  
* nvim-web-devicons
* nightfox.nvim
* lualine.nvim
* vim-gitgutter
* vim-fugative
* vim-startify
* popup.nvim
* plenary.nvim
* telescope
* comment.nvim
* luasnip
* nvim-lspconfig
* nvim-cmp
* cmp-cmdline
* cmp-path
* cmp_luasnip
